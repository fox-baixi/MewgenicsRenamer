# Mewgenics Cat Renamer (猫猫改名器)

很多玩家反映在玩《Mewgenics》的时候，由于游戏内不支持输入中文或者长字符，各种各样的默认随机洋文名字导致**很难分辨到底哪只猫是哪只猫**。在选种繁育、看技能和安排职业的时候看名字看花了眼。

由于游戏存盘数据的封包特殊性（LZ4引擎加密压缩），强行使用十六进制文本编辑器硬改中文非常容易导致数据错位或坏档。

因此，这个**傻瓜式无限制改名工具**诞生了！它能完美解开存档限制，帮你给猫自由地取任何中文（或任何长短的汉字/特殊符号）名字。

---

## 🚀 怎么用？
**下载现成工具：**
1. 去本页右侧的 [Releases](#) 里面下载最新的 `MewgenicsRenamer.exe`。
2. 双击打开。
3. 它会自动定好你的本地存档位置。**在列表里选猫，直接打新名字，点击 Apply！**
4. 进游戏看看，包爽。

*(工具非常安全，每次改名都会自动帮你备份一次原本的 `.bak` 存档文件，如果弄坏了可以直接改后缀救回来。)*

## ⚠️ 注意事项
**请务必不要在载入或处于游戏内（如布置房间、探索时）修改！**
- 这是因为改掉后游戏内存已经锁定了旧数据。
- 请在**退到游戏主界面 (Main Menu)** 或**关闭游戏**时再打开本工具进行改名编辑，然后重新读档进入！
